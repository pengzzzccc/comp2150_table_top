\section{概述}

本文档定义了 \textbf{Wolves of Wand Street} 的完整美术资产需求。这是一款面向3--4人的卡牌驱动区域控制桌游，共包含26张独特卡牌设计和3张卡背，合计 \textbf{29项美术资产}。

\subsection{主题与风格}

\begin{itemize}[leftmargin=*]
    \item \textbf{世界观设定：} 古典城邦与华尔街金融的融合。投资者-巫师在未被开发的新大陆上进行经济战争。
    \item \textbf{整体调性：} 掠夺性与趣味性并存。风格参考"华尔街之狼 × 中世纪商人公会"。
    \item \textbf{配色方案：} 金色、深蓝、暗红为主色，辅以奥术紫和森林绿。
    \item \textbf{卡牌尺寸：} 标准扑克牌尺寸（63mm $\times$ 88mm）或 playingcards.io 默认尺寸。
    \item \textbf{分辨率：} 最低300 DPI，PNG格式（透明背景或满版出血）。
\end{itemize}

\subsection{卡牌版面要求}

每张卡牌必须包含：
\begin{enumerate}[leftmargin=*]
    \item \textbf{卡牌名称}（顶部）
    \item \textbf{插画区域}（中央，约占卡面60\%）
    \item \textbf{效果文字}（底部）
    \item \textbf{费用}（仅限市场卡，右上角，以Gold标注）
    \item \textbf{花色/类型标识}（角落图标或颜色编码边框）
    \item \textbf{卡牌数量}（仅限抽牌堆卡，左下角，如 $\times$2）
\end{enumerate}

\section{交付要求}

\begin{itemize}[leftmargin=*]
    \item \textbf{文件格式：} PNG，最低300 DPI
    \item \textbf{卡牌尺寸：} 标准扑克牌（63mm $\times$ 88mm）或 playingcards.io 兼容尺寸
    \item \textbf{背景：} 满版出血（延伸至卡牌边缘）或透明背景
    \item \textbf{文字：} 所有卡名、效果、费用在印刷尺寸下必须清晰可读
    \item \textbf{颜色编码：} 各花色在视觉上必须明显区分（方块=红色、红心=红色、黑桃=深色、梅花=紫色）
    \item \textbf{风格统一：} 所有卡牌必须保持统一的画风、配色方案和排版风格
\end{itemize}
